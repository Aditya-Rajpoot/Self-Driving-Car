# Self-Driving-Car
This project is a Self-Driving Car Simulation built with Vanilla JavaScript and HTML5 Canvas. The system demonstrates how a car can learn to drive autonomously using Neural Networks and evolutionary learning techniques, without relying on external machine learning libraries.
The simulation uses ray-casting sensors to detect obstacles, processes the sensor data through a custom neural network, and controls the car's steering and movement decisions in real time.
Multiple AI cars are generated, and through mutation and selection, the best performing neural network is preserved and improved over time.

# Key Concepts Implemented
1.Neural Networks (from scratch)
2.Ray-Casting Sensors
3.Collision Detection
4.Evolutionary Learning (Mutation)
5.Autonomous Decision Making
6.Canvas-based Physics Simulation

# Technologies Used
1.JavaScript (Vanilla JS)
2.HTML5 Canvas
3.CSS
4.Neural Network Logic (Custom Implementation)
